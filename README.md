# flask-unittest-demo
A demo Flask project for unittesting socket.io API

## Installation
To be completed

## Running
1. Enter pipenv shell
```
pipenv shell
```
2. Run dev server
```
python3 run.py
```

## Testing
1. Export flask app
```
export FLASK_APP=flask-test.py
```
2. Start testing
```
flask test
```
